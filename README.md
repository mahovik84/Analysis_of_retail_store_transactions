# Анализ транзакций сети розничных магазинов.

## Цель проекта: 
 Выявить операционные аномалии, оценить влияние маркетинговой акции на прибыль и средний чек, а также визуализировать ключевые показатели бизнеса.

## Что сделано:

— Проведён анализ данных по транзакциям сети розничных магазинов.
— Изучена длительность операций и выявлены аномальные значения; интерпретированы с учётом бизнес-контекста.
— Проанализированы результаты маркетинговой кампании: оценено влияние акции на выручку, прибыль и средний чек.
— Рассчитаны коэффициент прироста и динамика ключевых метрик по периодам.
— Построены интерактивные дашборды и графики в Google Sheets, визуализирующие результаты анализа.

## Используемые инструменты: 
 Excel, Google Sheets, функции QUERY, сводные таблицы, визуализации (графики, дашборды).

## Результат: 
 Подготовлен аналитический отчёт и интерактивный дашборд, демонстрирующие влияние маркетинговой активности на финансовые показатели компании. "
